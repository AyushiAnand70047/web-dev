Units -> Absolute(px), Relative(%,em,rem,vh,vw)

1em means same as parent size, 2em means twice of parent size,...

rem is with respective to root element(html is root)

vh -> viewport height (size based on screen height)

vw-> viewport width(size based on screen width)

Positioning -> static(default), fixed(fix position on screen, not change on scroll), absolute(absolute with nearest relative parent, if no relative parent exist then it follow browser window), sticky(relative on same screen and fixed when screen changes on scroll(touched top))
